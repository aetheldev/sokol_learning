# VFX Learning Path

This folder is exercise-first.

Goal:
- learn how to make games feel alive
- learn glow, particles, burning, impact, and beam effects
- practice effects you can later rebuild inside `sauce/`

How to use it:
- start with each `LESSON.md`
- only read exact parts of the solution it points to
- close solution
- write your own `main.odin`

Runnable answers live in:
- `learn/solutions/vfx/`

Production goal after this path:
- `learn/production_with_sauce/09_visual_effects_roadmap.md`
